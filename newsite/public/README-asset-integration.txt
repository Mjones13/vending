# Asset Integration for Smarter Vending Replica

This directory will contain all static assets (images, icons, logos) used throughout the site.

## Required Assets (to match https://smartervendinginc.com/)
- Company logos (Disney, Sheraton, Marriott, Toyota, Audacy, Romeo Power, Splitsville, Fresh N Lean, Din Tai Fung)
- Hero/landing images
- Service icons (Mini Markets, Coffee Services, Vending Machines)
- Product images (coffee, snacks, etc.)
- UI icons (menu, social, etc.)
- Any additional images from the reference site

## Integration Plan
- Add all images in this directory and reference them in the appropriate pages/components.
- Use descriptive filenames (e.g., disney-logo.png, hero-bg.jpg, coffee-icon.svg).
- Optimize images for web (compressed, appropriate resolution).
- Update this README as assets are added.
